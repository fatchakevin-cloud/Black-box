Petit site statique en HTML/CSS avec trois pages : `index.html`, `services.html`, `contact.html`.

Ouvrir localement :

- Double-cliquer sur `index.html` dans l'explorateur de fichiers
- Ou lancer un serveur local (ex. PowerShell) :

```powershell
# depuis le dossier du projet
python -m http.server 8000
# ou, si vous avez PowerShell 7+ et le module 'SimpleHTTPServer' ou similaire
# puis ouvrir http://localhost:8000
```

Modifiez les fichiers dans `css/` et les pages `.html` selon vos besoins.
